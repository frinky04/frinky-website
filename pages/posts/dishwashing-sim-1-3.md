# Patch Notes

## Added

- Flashlight now acts like an upgrade, equip it, and then toggle with "T" hands-free
- Hint system to provide valuable info that can't quite be communicated diegetically
- 20 Achievements for you to unlock
- Trophy item, costs 15k, something big to aim for.
- Zoom feature: Hold middle mouse while using a computer to zoom closer to the text.
- UI Scale setting to increase the scale of text and images.

## Changed

- Updated the Research Table's description to be more accurate
- Book UI overhaul / font change
- Internet browser now displays a warning that it is indeed a real browser
- Waking up will now bump low stats (Health, Food, Water) up to at least 20% (Enough to avoid a soft lock)
- Conveyor walls can now be toggled on or off
- Increased rate of hostile encounters.
- ||The Jester now spawns independently of other encounters. Watch for the beartraps when running from other entities.||
- ||Made it easier to tame stalkers||

## Fixes

- Fixed some misc. spelling and grammar issues
- Adjustments to options UI formatting
- Player movement no longer collides with conveyor walls
- Fixed in-game screens being cut off when playing with widescreens
- Hardened save loading to prevent launch crashes from corrupted saves. (maybe, needs testing)
- Sometimes your character would keep walking when you were using a computer.
- Grabbed object mass interpolation now updates more often. (Fixes crates having desynced mass)
- ||Screwdriver is now delivered as a gift||
